## การเพิ่มโมดูลคณิตศาสตร์ในภาษาต่าง ๆ

Ruby

```ruby
include Math
```

C

```c
#include <math.h>
```

Java

```java
import java.lang.Math.*;
```

Python

```python
import math
#or
import cmath # สำหรับจัดการจำนวนเชิงซ้อนจะมีบวกจำนวนเชิงซ้อนต่อท้าย 
#เช่น print(cmath.sqrt(9)) # Output: (3+0j)
#สำหรับเอกสารนี้อธิบายเพียง math
```
